为了简单期间，用户信息没有存放在数据库，而是放在内存中。初始化时用户名密码：admin/admin

最简单的存放登录token的方式：session，利用servlet的实现。

如果需要更进一步保存用户登录态，就需要cookie。

关于密码：
不要明文在网路上传输，不要db保存明文密码，
不要把密码或密码加密放到cookie，要用一次一密。
登录过程应该使用https。
